# 🦐 Tampermonkey Scripts

个人自用油猴脚本合集，提升日常上网体验。

## 目录结构

```
tampermonkey-scripts/
├── src/           # 脚本源码
│   ├── utils/     # 公共工具函数
│   └── ...        # 各功能脚本
├── README.md
└── ...
```

## 使用方式

1. 安装 [Tampermonkey](https://www.tampermonkey.net/) 浏览器扩展
2. 打开对应脚本的 `.user.js` 文件原始链接
3. Tampermonkey 会自动识别并提示安装
